# Project Rules

## 배포 정책 (Deployment Policy)

**배포는 사용자가 명시적으로 요청할 때만 수행합니다.**

다음 조건을 모두 충족해야 배포를 진행할 수 있습니다:
1. 사용자가 "배포", "deploy", "Vercel에 올려줘" 등 명확한 배포 요청을 함
2. 코드 수정이나 기능 구현 완료 후 자동으로 배포하지 않음
3. 배포 전 사용자에게 배포 여부를 확인받음

### 금지 사항
- 코드 변경 후 "배포까지 해드릴까요?" 제안 금지
- 빌드 성공 후 자동 배포 금지
- 사용자 확인 없이 `vercel --prod` 실행 금지
